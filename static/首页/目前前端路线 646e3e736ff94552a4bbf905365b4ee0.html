目前前端路线

学习初期

  1. 《编码——隐匿在计算机软硬件背后的语言》，看到「第 12 章 二进制加法器」。这本书让我理解什么是计算机。
  1. 阮一峰的 JavaScript 教程，https://wangdoc.com/javascript/
  1. 阮一峰的 ES6 教程，http://es6.ruanyifeng.com/ ，看完这两个基本就知道怎样写 JS 代码了。
  1. 然后需要学习另一门语言 java
  1. 接下来看算法《我的第一本算法书》宫崎修一 石田保辉，看到深度优先搜索
  1. 练手，去 leetcode 刷题

进入实践

  1. 看《CSS 揭秘》，提升自己对 CSS 的掌握。
  1. 开始做项目，比如自己做个人主页，画板、做个计算器、做个扫雷，反正就是把见过的东西都自己实现一次，能不能做成功不要紧，关键是过程。做完三四个这样的项目之后，开始学习框架
  1. Vue ，把官方文档一字不漏地看完
  1. 看完之后把之前的项目用 Vue 或 React 重写
  1. 开始做更大的项目，比如仿一个 Todo MVC，仿一个购物车应用，总之把你见过的东西都拿过来用 Vue 做做看。
  1. 提升

在做的过程中会学会科学上网、Stackoverflow、GitHub、Git 等。

会无数次地查看 API 文档，并逐渐记住常用的 API。

了解 babel webpack typescript http，把文档看一下就直接干。

这个过程会卡壳很多次，请自己想办法搞定。
